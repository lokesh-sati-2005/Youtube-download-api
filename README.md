# YouTube Download API

A Python backend service built with Flask that allows users to download YouTube videos using a URL. The project is designed to explore REST API development, request handling, input validation, and file processing workflows.

## Features

- Download YouTube videos by URL
- Fetch available video resolutions
- Input validation
- Structured error handling
- REST API endpoints
- Modular backend architecture

## Tech Stack

- Python
- Flask
- yt-dlp
- REST APIs
